PhonePe Business Insights Dashboard

Project Overvie

Developed an interactive Power BI dashboard to analyze digital payment transactions using the PhonePe Pulse dataset (2018–2022). The dashboard provides comprehensive insights into transaction volume, transaction value, user behavior, regional performance, and growth trends across India.

Objectives

Analyze transaction trends over multiple years.
Identify high-performing transaction categories and regions.
Examine quarterly growth patterns and seasonal variations.
Provide data-driven insights for business decision-making.

Key Insights

Significant growth in digital transactions from 2018 to 2022.
Strong contribution from merchant and peer-to-peer payment categories.
Regional variations in transaction adoption and usage.
Consistent quarterly growth indicating increasing digital payment penetration.
